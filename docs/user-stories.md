# User Stories

## Main Route Flow

As a user, I want to speak a route request so that RoadMuse can build a route without me filling forms.

As a user, I want a text fallback so that RoadMuse works even when browser speech recognition fails.

As a user, I want to see what RoadMuse understood so that I can trust the route plan before opening another navigator.

## Preferences

As a user, I want to enter plain-English preferences so that I can describe how I actually like to drive.

As a user, I want each preference validated so that I know whether RoadMuse can honor it.

As a user, I want preferences like “when driving from A to B prefer C” to apply only in the right context.

## External Navigators

As a user, I want to choose Google Maps, Waze, Apple Maps, HERE WeGo, Organic Maps, or GPX export.

As a user, I want RoadMuse to warn me when my selected navigator cannot preserve waypoints or exact route shape.

As a Waze user, I want RoadMuse to open Waze search/navigation with supported avoid settings.

## Competitor-Parity Features

As a user, I want to search by need, like “coffee and restroom,” not just by business name.

As a user, I want along-route search so I can add stops without manually browsing the map.

As a user, I want multi-stop optimization for errands.

As a user, I want route explanation so I know why RoadMuse picked a specific route.
